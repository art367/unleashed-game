# 🐕 UNLEASHED! — Dog Park Chaos

**Your dog's slipped the lead... chaos is about to happen!**

An endless runner browser game where YOU are the naughty dog causing mayhem in the park while your owner chases behind shouting increasingly desperate things.

## 🎮 [PLAY NOW!](https://art367.github.io/unleashed-game/)

## Features
- 🐕 **5 unique dog breeds** — Labrador, Greyhound, Terrier, Pug, Great Dane
- 😱 **Panicking owner** with hilarious voice commentary (male & female)
- 💩 **Poo mechanics** — bag it or leave it, the eternal dilemma
- 🎯 **Dog vs Owner scoring** — who wins the chaos battle?
- 🧸 **Squeaky teddy bear** — "who's my little baby!"
- 👮 **Park keepers** — dodge them or it's BUSTED!
- 🌳 **Tree wee** — because every dog has to mark their territory
- 🍗 **Drumstick stealing** — naughty boy, leave it!
- 📲 **PWA** — install on your phone like a real app
- 📤 **Share your chaos rating** with friends

## Tech
- Single-file HTML5 Canvas game
- Browser speechSynthesis for owner voice
- Web Audio API for SFX
- Progressive Web App (installable)
- No frameworks, no dependencies, no excuses
